# React Weather Web App

## Live demo
See a live demo [here](https://react-weather-web-app.surge.sh/)

## Build With

* React
* React Hooks
* Yarn
* create-react-app
* OpenWeather Api
* SweetAlert 2
* Axios
* Bootstrap 5

## How to install

```
yarn
```

## How to use

Development server:
```
yarn start
```

Build:
```
yarn build
```
